# Blynko2

A Plinko game built with Python, Pygame, and Pymunk.

## Controls

| Input | Action |
|-------|--------|
| Space | Drop ball from center |
| Left-click (top bar) | Drop ball at cursor position |
| Left-click (play area) | Place a peg |
| Right-click | Remove nearest peg |
| R | Reset ball |
| C | Clear all pegs |
| Escape | Quit |

## Scoring

Buckets across the bottom award points based on position:

| 50 | 100 | 200 | 500 | 200 | 100 | 50 |

## Setup

```bash
pip install -r requirements.txt
python plinko.py
```
